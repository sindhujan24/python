### Declaring variables
variable_name = "value"
example: a = 10
         name = "Mili"
         
## Datatypes
Python has the following data types,
- Text Type:	str
- Numeric Types:	int, float, complex
- Sequence Types:	list, tuple, range
- Mapping Type:	dict
- Set Types:	set, frozenset
- Boolean Type:	bool
- Binary Types:	bytes, bytearray, memoryview
